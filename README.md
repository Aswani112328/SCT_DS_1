# Student Data Analysis Dashboard

## Project Overview

This project presents a Student Data Analysis Dashboard that provides insights into student demographics using visual analytics. The dashboard helps in understanding the distribution of students by gender, age, and school through clear and interactive visualizations.
The analysis is useful for academic institutions to evaluate enrollment patterns and demographic trends.

##  Dataset Information
*File name: `student_data.csv`
*Data includes:

  * Student age
  * Gender (Sex)
  * School (GP, MS)

The dataset is used as the primary source for creating the dashboard visualizations.

## Dashboard Components

### 1. Gender Distribution of Students

* Displays the count of students based on gender.
* Helps identify gender balance across the dataset.

### 2. School-wise Distribution of Students

* Shows the number of students enrolled in each school.
* Schools included:

  *GP
  *MS

### 3. Age Distribution of Students

* Visualizes student count across different age groups.
* Helps understand the age range and concentration of students.

1. Open Tableau Desktop.
2. Load the file `Book1.twbx`.
3. Interact with the dashboard to explore student data insights.


##  Key Insights

* Majority of students belong to the GP school.
* Student ages are concentrated within a specific range.
* Gender distribution is relatively balanced.


##  Conclusion

This dashboard provides a simple yet effective way to analyze student data visually. It can be extended by adding performance metrics, attendance data, or year-wise comparisons.
